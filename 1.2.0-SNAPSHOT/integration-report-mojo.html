scoverage:integration-report

Note:This goal should be used as a Maven report.

Full name:

org.scoverage:scoverage-maven-plugin:1.2.0-SNAPSHOT:integration-report

Description:

Generates code coverage by unit and integration tests report in forked scoverage life cycle.

In forked scoverage life cycle project is compiled with SCoverage instrumentation and unit and integration tests are executed before report generation.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.1.0.
  • Invokes the execution of the lifecycle phase verify prior to executing itself.
  • Executes in its own lifecycle: scoverage.

Optional Parameters

Name Type Since Description
aggregate boolean 1.1.0 Aggregate SCoverage reports.

Default value is: false.
User property is: scoverage.aggregate.
encoding String 1.2.0 The file encoding to use when reading Scala sources.

Default value is: ${project.build.sourceEncoding}.
User property is: encoding.
skip boolean 1.0.0 Allows SCoverage to be skipped.

Default value is: false.
User property is: scoverage.skip.
