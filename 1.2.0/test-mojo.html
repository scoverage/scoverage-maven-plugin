scoverage:test

Full name:

org.scoverage:scoverage-maven-plugin:1.2.0:test

Description:

Executes forked scoverage life cycle up to test phase.

In forked scoverage life cycle project is compiled with SCoverage instrumentation and unit tests are executed.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.2.0.
  • Invokes the execution of the lifecycle phase test prior to executing itself.
  • Executes in its own lifecycle: scoverage.

Parameters

(no parameters)
